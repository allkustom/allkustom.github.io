Remnant Of Art, Art Of Remnant

Date: 05/10/2023
Medium: Eco epoxy resin / Leftover material / Acrylic
Size: 11.8*11.8*23 inch / 30*30*60 cm


Description:
'Every art has its remnants'.
Honestly, is it possible to create the intended artwork using the leftover material? It is extremely limited. Therefore, it can't be used as a material for artwork again and is discarded. It happens to every artist in the world, not only us. We need a system that can recycle them in an eco-friendly practical way. Using epoxy resin to fix remnants and seal gaps in between them. And forming as furniture. This project isn't going to end in just one piece. It will be a prototype to determine whether the implementation of the system will be possible.
