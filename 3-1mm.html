Date: 05/17/2023
Medium: Unity / Blender / Photoshop / Illustrator / After Effect / Audtion
Size: Playtime - 12min


Click here to download the game


Description:
My fist Unity & Blender Project
(100% handmade assets, except music)

'Did I live a life without regrets?'
A man who made the imperfect time-machine request you to conclude his life. It's unable to rewind the time, but able to rewrite the speed of the memory. A happy and pleasant memory passes so fast. An hour feels like a minute. But, a boring and sad memory feels like endless. A minute turns into an hour. Which memory is precious? By inserting the memory into the time module, you can conclude his life. Is it possilbe to live a life without regrets? It's up to you.

Game control:
Move: Wasd
Dash: Shift
Interact: Mouse left-click(with highlighted object)

Language:
English & Korean
